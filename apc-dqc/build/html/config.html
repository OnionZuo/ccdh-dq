config 模块¶

变量

类型

描述

DEVCONN

dict

元数据库的连接信息，同元数据库连接信息配置表ID=1的连接信息

EMAILCONFIG

dict

邮件通知配置，包括发件邮箱账号、密码、服务器端口，收件人等

EMAIL_FAIL

str

任务失败邮件模板，默认包含输出日志文件内容

EMAIL_SUCCESS

str

任务成功邮件模板，默认包含任务成功失败明细和输出日志文件内容

EMAIL_TMPL

str

通用邮件模板，可以使用CSS语言自定义格式

MASTERTBS

dict

元数据库的连接信息，即元数据库 连接信息配置表 ID=1的连接信息

dict

元数据库的连接信息，即元数据库 连接信息配置表 ID=1的连接信息

str

程序连接SQL Server使用的ODBC驱动，参见 ODBC Driver for SQL Server

TZ

str

程序输出日志中时间的时区，参见：pytz
